<h2>Data Preprocessing on Titanic Dataset</h2>

<h3>Aim</h3>
<p>
To implement data preprocessing techniques such as data cleaning, data integration,
data transformation, data reduction, and data discretization using Python.
</p>

<h3>Dataset</h3>
<p>
Titanic Dataset containing numerical and categorical attributes with missing values and outliers.
</p>

<h3>Tools Used</h3>
<ul>
    <li>Windows 7 or above</li>
    <li>Anaconda Distribution</li>
    <li>Python 3.7+</li>
    <li>Jupyter Notebook</li>
</ul>

<h3>Preprocessing Steps</h3>
<ul>
    <li><b>Data Cleaning:</b> Handling missing values, removing duplicates and outliers (IQR method)</li>
    <li><b>Data Integration:</b> Splitting and merging train and test datasets</li>
    <li><b>Data Transformation:</b> Encoding categorical data and scaling numerical data</li>
    <li><b>Data Reduction:</b> Feature selection, PCA, and sampling</li>
    <li><b>Data Discretization:</b> Converting Age into Young, Adult, and Senior groups</li>
</ul>

<h3>What We Learned</h3>
<ul>
    <li>Importance of data preprocessing in machine learning</li>
    <li>Handling missing values and outliers</li>
    <li>Difference between normalization and standardization</li>
    <li>Use of PCA for dimensionality reduction</li>
    <li>Role of discretization in data analysis</li>
</ul>

<h3>Conclusion</h3>
<p>
Data preprocessing improves data quality and prepares the dataset for effective
machine learning and data mining tasks.
</p>
